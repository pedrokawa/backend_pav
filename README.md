# Backend PAV

Esta é o backend para projetos da PAV, desenvolvida em Node.js. O projeto utiliza o Prisma como ORM para a modelagem e gerenciamento do banco de dados e já está configurado para deploy na plataforma Vercel.

## Tecnologias Utilizadas

- **Node.js** & **JavaScript**
- **Prisma** (ORM)
- **Vercel** (Hospedagem e Serverless)
